## What I build

I specialize in multi-agent systems using **LangGraph**, **CrewAI**, and **MCP** — with a focus on production reliability, not proof-of-concept demos. My work sits at the intersection of AI orchestration and real business operations.

**Current focus areas:**
- Multi-agent e-commerce operations (inventory, pricing, fraud)
- MCP tooling and infrastructure for agentic applications
- Agent evaluation frameworks (benchmarking, adversarial testing, cost optimization)
